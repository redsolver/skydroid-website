How to install

Wait for the download to finish, then click on the downloaded APK file to start the installation process. You may be prompted to enable "Installation from Unknown Sources" in the process.

Older Android devices

If you are on an older Android version, you may have to enable "Unknown Sources" manually in your Settings. To do this, you have to open your "Settings" app, then select "Security" and finally enable "Unknown Sources" there. You should be able to install the downloaded APK file after that.
